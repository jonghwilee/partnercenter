### 1.1 목적

- 파트너센터 FRD(`FRD_v0.2.9.md`) **FR-PRE01** 이 참조하는 **상태 매트릭스 및 자격 조건의 정본**이다.

| **운영자** | 비즈니스센터 내부 운영 담당자 | 이용 제한 상태 변경 권한 보유 |

## 3. 브랜드 프로필 도메인

### FR-BP01 — 브랜드 프로필 생성 **(P0)**

#### BP01-1. 생성 자격

- **약관동의 + 이메일 인증을 완료한 비즈니스 회원**만 브랜드 프로필을 생성할 수 있다.
- 아래 경우 생성 불가:
  - 이메일 인증이 완료되지 않은 계정
  - 이용 제한 상태의 계정

#### BP01-2. 수량 제한

| 제한 항목 | 제한값 | 비고 |
|-----------|--------|------|
| 회원 1명이 **생성** 가능한 브랜드 프로필 수 | **5개** | 삭제 완료된 것은 미포함(FR-BP06) |
| 회원 1명이 **관리** 가능한 브랜드 프로필 수 | **25개** | 생성·초대로 관리자가 된 경우 모두 합산. 삭제 완료는 제외 |
| 브랜드 프로필 1개당 등록 가능한 **관리자 수** | **30명** | 주관리자 포함 |

#### BP01-3. 명칭 및 정보

- 브랜드 프로필 이름은 **서비스 내 중복 허용**한다.
- 아래 명칭은 사용 불가:
  - 욕설이 포함된 명칭
  - 타인의 상표 등 권리를 침해하는 부적절한 명칭

#### BP01-4. 심사 절차

- 브랜드 프로필은 **대표자·직원·대행사**가 **별도의 심사 과정**을 통해 생성한다.
- 심사 흐름은 FR-BP02(인증) → FR-BP03(상태 관리) → FR-BP05(승인)를 따른다.

**수용 기준**

- 수량 제한 초과 시 생성 버튼 비활성 또는 명확한 안내 메시지를 제공한다.
- 이용 제한 계정으로 생성 시도 시 진입 차단 및 안내를 제공한다.

---

### FR-BP02 — 브랜드 프로필 인증 **(P0)**

#### BP02-1. 사업자 인증

- 브랜드 프로필 생성을 위해 **사업자 인증**이 필수다.
- 국세청에 사업자 정보가 등록된 상태로 **「영업중」** 상태여야 한다.
- **「휴업」 또는 「폐업」** 상태인 경우 브랜드 프로필 생성 불가.

#### BP02-2. 사업자와의 관계 인증(신청인 정보 확인)

신청 가능 대상은 해당 사업자의 **대표자·직원·대행사**에 한하며, 관계별로 아래 인증 서류를 제출한다.

| 신청인 유형 | 제출 서류 |
|-------------|-----------|
| **대표자** (개인/법인) | 사업자등록증 |
| **직원** | 재직증명서 |
| **대행사** | 사업자등록증 + 광고주 사업자등록증 + 대행계약서 |

**수용 기준**

- 사업자 인증 실패(휴·폐업 등) 시 생성 불가 사유를 명확히 안내한다.
- 관계 인증 서류 미제출·형식 불일치 시 반려 처리한다(FR-BP03 반려 상태).

---

### FR-BP03 — 브랜드 프로필 상태 관리 **(P0)**

#### BP03-1. 상태 정의

브랜드 프로필은 아래 7개 상태를 가진다. 상태 전이는 심사 결과·운영자 조치·삭제 요청에 따라 결정된다.

| 상태 | 설명 |
|------|------|
| **(심사)요청** | 최초 생성 신청 후 심사 대기 중 |
| **반려** | 심사에서 반려됨. 재심사 신청 가능 |
| **재심사** | 반려 후 재심사 요청 또는 사업자 정보 수정 후 재심사 진행 중 |
| **승인** | 심사 통과. 파트너센터 접근 가능 상태(FR-PRE01) |
| **삭제 대기** | 삭제 요청 후 처리 중 |
| **삭제 완료** | 삭제 완료. 비즈니스센터 홈에서 비노출 |
| **이용 제한** | 운영자가 이용 제한 처리한 상태 |

#### BP03-2. 상태별 기능 가능 여부 매트릭스

주관리자인 경우에 한해 아래 기능이 허용된다.

| 브랜드 프로필 상태 | 브랜드 프로필 정보 접근 | 프로필 이름 수정 | 사업자 정보 수정 | 재심사 요청 | 프로필 삭제 |
|--------------------|------------------------|-----------------|-----------------|------------|------------|
| **(심사)요청** | 가능 | 불가능 | 불가능 | 불가능 | 불가능 |
| **반려** | 가능 | 불가능 | 가능 (주관리자) | 가능 (주관리자) | 가능 (주관리자) |
| **재심사** | 가능 | 불가능 | 불가능 | 불가능 | 불가능 |
| **승인** | 가능 | 가능 (주관리자) | 가능 (주관리자) | 불가능 | 가능 (주관리자) |
| **삭제 대기** | 불가능 | 불가능 | 불가능 | 불가능 | 불가능 |
| **삭제 완료** | 불가능 | 불가능 | 불가능 | 불가능 | 불가능 |
| **이용 제한** | 불가능 | 불가능 | 불가능 | 불가능 | 불가능 |

**수용 기준**

- 상태별 허용 기능은 **서버 API에서도 동일하게 강제**한다(프론트 표시 제어만으로 불충분).
- 삭제 완료 상태의 브랜드 프로필은 비즈니스센터 홈에서 비노출한다.

### FR-BP04 — 브랜드 프로필 재심사 **(P0)**

- 아래 두 경우에 재심사를 진행한다:
  1. 심사 **반려** 후 **재심사를 요청**한 경우
  2. **「사업자 정보 수정」** 을 요청한 경우
- **재심사 상태**일 경우, 아래 기능은 모두 불가하다:
  - 프로필 정보 수정
  - 사업자 정보 수정
  - 브랜드 프로필 삭제

**수용 기준**

- 재심사 진행 중임을 사용자가 화면에서 확인할 수 있어야 한다.
- 재심사 결과(승인/재반려)는 사용자에게 알림 등으로 전달한다(알림 채널은 운영 정책에서 확정).

---

### FR-BP05 — 브랜드 프로필 승인 **(P0)**

#### BP05-1. 사업자 정보 검증 (국세청 API)

사용자가 입력한 다음 항목을 **국세청 API**와 대조한다.

- 사업자번호
- 상호명
- 대표자명
- 개업일
- 사업자등록증

| 결과 | 처리 |
|------|------|
| **성공** (모든 항목 일치) | 다음 단계로 이동 |
| **실패** (1개라도 불일치) | 실패 처리. OCR 배치로 사업자등록증 정보 추출 후 재검증 또는 반려 |

#### BP05-2. OCR 사업자등록증 정보 추출

- 사업자등록증 이미지에서 정보를 **OCR(배치)**로 자동 추출한다.
- 추출 결과를 국세청 API 검증에 활용하거나, 심사 담당자 검토에 제공한다.

**수용 기준**

- 국세청 API 실패 시 사용자에게 명확한 실패 사유와 재시도 방법을 안내한다.
- OCR 처리 결과는 내부 심사 흐름에서 확인 가능해야 한다(운영자 화면 범위는 별도 명세).

---

### FR-BP06 — 브랜드 프로필 삭제 **(P0)**

#### BP06-1. 삭제 전 제한조건 확인

삭제 요청 전, **하위 비즈니스 서비스의 삭제 제한조건**을 확인한다.

| 서비스 | 삭제 제한 조건 |
|--------|---------------|
| **광고센터** | 심사 대기 / 심사 중 / 승인 / 진행 중인 광고 캠페인이 있는 경우 |

- 제한조건이 없을 경우에만 삭제 요청을 수행한다.

#### BP06-2. 삭제 처리

- 삭제 요청 완료 시 브랜드 프로필이 **즉시 삭제**된다.
- 이후 시점부터 **관리자 전원 해당 브랜드 프로필에 접근 불가**.
- 삭제된 브랜드 프로필의 데이터는 **7일간 보관**한다. 7일 이내에는 **고객센터를 통해 삭제 철회** 가능.

#### BP06-3. 삭제 후 정책

- 삭제 완료 후 **동일한 이름으로 새 브랜드 프로필 생성 가능**.
- 삭제된 브랜드 프로필은 **관리 중인 브랜드 프로필 개수(25개)에 포함되지 않는다**.

**수용 기준**

- 삭제 제한조건에 걸리는 경우, 사용자에게 해당 제한 내용을 명시하고 삭제 요청을 차단한다.
- 삭제 완료 시 파트너센터 접근은 즉시 차단된다(**FR-PRE01** 상태 = 삭제 완료).

---

### FR-BP07 — 브랜드 프로필 이용 제한 **(P0)**

- **생성 불가 업종** 또는 **기타 운영정책을 위반한 업종**에 대해 이용 제한이 필요한 경우, **운영자**가 「이용 제한」 상태로 변경한다.
- 이용 제한 상태에서는 **브랜드 프로필에 접근 불가**.

**수용 기준**

- 이용 제한 처리는 운영자 권한으로만 수행 가능하다.
- 이용 제한 시 해당 브랜드 프로필에 연결된 파트너센터 접근도 즉시 차단된다(**FR-PRE01** 상태 = 이용 제한).

---

### FR-BP08 — 브랜드 프로필 탈퇴 **(P0)**

#### BP08-1. 부관리자 탈퇴

- **부관리자**는 자발적으로 해당 브랜드 프로필에서 탈퇴할 수 있다.

#### BP08-2. 주관리자 탈퇴

- **주관리자**가 탈퇴하기 위해서는 소속된 브랜드 프로필에 **본인을 제외한 1명 이상의 주관리자**가 존재해야 한다.
- 주관리자는 언제든지 **브랜드 프로필 구성원(부관리자 포함)을 강제 탈퇴** 시킬 수 있다.

#### BP08-3. 탈퇴 후 데이터 처리

- 프로필에서 탈퇴가 완료되는 경우, 관련 법령 및 개인정보처리방침에 따라 보유하는 정보를 제외한 **사용자 데이터는 즉시 삭제**된다.

**수용 기준**

- 주관리자가 다른 주관리자 없이 탈퇴 시도 시 안내 메시지와 함께 차단한다.
- 강제 탈퇴 처리 시 해당 구성원에게 알림을 제공한다(알림 채널은 운영 정책에서 확정).

---

### FR-BP09 — 사업자 정보 변경 **(P1)**

사업자 정보 변경이 필요한 경우, 아래 **변경 사유 6종** 중 하나를 선택하여 요청한다. 사유에 따라 추가 서류·처리 흐름이 달라질 수 있다.

| 변경 사유 | 설명 |
|-----------|------|
| **양도양수** | 기존 사업자가 보유한 사업의 관리를 다른 사업자에게 이전하는 경우. 양도양수 범위는 부분 양도(일부 사업), 특정 서비스/브랜드 이전도 포함 가능 |
| **인수합병 (M&A)** | 기존 사업자의 소유권이 다른 사업자로 이전되거나, 두 개 이상의 사업체가 하나로 합쳐지는 경우. 기업의 확장, 구조 조정, 경영 효율화 목적으로 이루어질 수 있으며, 인수합병의 형태에 따라 포괄/부분 인수합병으로 구분 |
| **과세 변경** | 기존 사업자의 과세 유형이 변경(사업자등록번호 변경)되는 경우. 개인사업자가 법인사업자로 전환되거나, 면세사업자가 과세사업자로 전환되는 경우, 사업자 단위 과세 적용에 해당 |
| **지자체 및 관계사 사업자 변경** | 지방자치단체(지자체) 또는 관계사가 운영하는 사업자의 사업자번호가 변경되는 경우. 운영 주체의 변경, 위·수탁계약 변경, 사업 구조 조정 등으로 인해 발생 |
| **휴폐업** | 사업자가 일정 기간 동안 사업을 중단하거나, 사업 운영을 폐업 신고한 후 다른 사업자로 변경하는 경우. 변경사업자의 대표자명, 업태, 종목, 주소 등 기존 사업자의 정보와 유사해야 함 |
| **기타** | 사업자 오등록 등 명변 등에 해당하지 않는 사례인 경우, 기타 항목으로 사업자번호 변경을 요청할 수 있음 |

**수용 기준**

- 변경 사유 선택 시 해당 사유에 맞는 제출 서류 안내를 제공한다.
- 변경 요청 후 재심사 흐름(FR-BP04)을 따른다.

## 4. 파트너센터 연동 지점

### 4.1 브랜드 프로필 상태 API

- 비즈니스센터는 파트너센터가 소비하는 **브랜드 프로필 상태 조회 API**를 제공한다.
- API 응답값은 FR-BP03 §BP03-1에서 정의한 7개 상태 중 하나를 반환한다.

### 4.2 파트너센터 진입 차단 정책

파트너센터는 **FR-PRE01** 기준으로 브랜드 프로필 상태를 확인하고, **「승인」 상태인 경우에만 정상 진입을 허용**한다.

| 브랜드 프로필 상태 | 파트너센터 처리 |
|--------------------|-----------------|
| 없음 (미생성) | 비즈니스센터 브랜드 프로필 생성 페이지로 이동 안내 |
| (심사)요청 / 재심사 | 심사 대기 안내 화면 표시, 파트너센터 본 화면 진입 차단 |
| 반려 | 반려 사유 안내 + 재심사 신청 링크 제공, 진입 차단 |
| **승인** | **파트너센터 정상 진입 허용** |
| 삭제 대기 / 삭제 완료 | 브랜드 프로필 재생성 안내, 진입 차단 |
| 이용 제한 | 이용 제한 안내, 진입 차단 |

- 세부 UX(차단 화면 문구, CTA 버튼, 비즈니스센터 이동 URL 등)는 파트너센터 FRD **FR-PRE01** 이 정본이다.
- 브랜드 프로필 삭제(FR-BP06) 또는 이용 제한(FR-BP07) 발생 시, 파트너센터 세션·접근 권한을 즉시 회수한다.
