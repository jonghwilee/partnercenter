# CJ ONE B2B — UI kit

Hi-fi recreation of the CJ ONE B2B Admin desktop console. Built from the
Figma file `[PC] B2B Design System v1.1.1.fig`. Contains a clickable
prototype that walks through:

1. **Login** — branded sign-in screen for partner staff.
2. **Dashboard** — KPI tiles, sales chart, partner list.
3. **Settlement** — table view with filters, pagination, status badges.
4. **Partner detail** — drawer-style detail page.
5. **Modal flows** — confirm dialog, snackbar.

Open `index.html` to walk through the prototype. Components are factored
inline (`<script type="text/babel">`) to keep the example readable.
